# Flipkart Mobile Data Scraper

A Python-based web scraping project that extracts mobile phone details from
Flipkart and stores them in a CSV file for analysis.

## Tech Stack
- Python
- Requests
- BeautifulSoup
- Pandas

## Features
- Scrapes mobile name, price, rating, image URL, and product link
- Stores scraped data into a CSV file
- Beginner-friendly and easy to understand

## How to Run
```bash
pip install requests beautifulsoup4 pandas
python scraper.py
